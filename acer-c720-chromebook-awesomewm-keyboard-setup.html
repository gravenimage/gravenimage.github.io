A simple journal : Acer C720 Chromebook AwesomeWM keyboard setup

Sun 21 December 2014

The Acer C720 is a lovely little web-browsing machine, and with Crouton can potentially be useful to a geekier audience. But with a limited screensize and a poor trackpad, it's sensible to go down the tiling WM route. Crouton details instructions for using AwesomeWM which worked out of the box.

Almost.

I've used emacs for years and have the muscle-memory to prove it. Specifically, I have the memory linked to a swapped control and capslock key. The capslock key which is now mod4 and controls AwesomeWM shortcuts.

Because this specific Chromebook (or is it all of them?) doesn't have the Windows key, the only available key to use for WM-commands is the alt gr key. It's a bit un-ergonomic, but a lot less than resetting years of experience. Here's the .Xmodmap file to do both the remapping of the search key to ctrl, and alt gr to mod4. Set this from your .xinitrc or just run it in the first console you create.

clear control
clear mod4
clear mod1
keycode 133 = Super_L
keycode 108 = Super_R
keycode 64 = Alt_L
add control  = Control_L Control_R Super_L
add mod4 = Hyper_L Super_R
add mod1 = Alt_L
